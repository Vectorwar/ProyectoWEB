Glosario de Términos Web

Definiciones de conceptos clave en desarrollo web

API

Application Programming Interface - Conjunto de reglas y protocolos que permite que diferentes aplicaciones se comuniquen entre sí.

CSS

Cascading Style Sheets - Lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML.

CRUD

Create, Read, Update, Delete - Operaciones básicas de gestión de datos en bases de datos y aplicaciones.

DOM

Document Object Model - Interfaz de programación para documentos HTML que representa la estructura del documento como un árbol de objetos.

Framework

Estructura de software que proporciona funcionalidades genéricas que pueden ser utilizadas para desarrollar aplicaciones.

HTML

HyperText Markup Language - Lenguaje de marcado estándar para crear páginas web.

JavaScript

Lenguaje de programación interpretado que permite crear contenido dinámico e interactivo en páginas web.

JSON

JavaScript Object Notation - Formato ligero de intercambio de datos fácil de leer y escribir para humanos y máquinas.

Responsive Design

Diseño web que se adapta automáticamente a diferentes tamaños de pantalla y dispositivos.

UI

User Interface - Interfaz de usuario, el espacio donde las personas interactúan con una aplicación.

UX

User Experience - Experiencia del usuario, cómo se siente una persona al usar un producto o servicio.

Variable

Contenedor con nombre que almacena un valor en programación, permitiendo su reutilización y manipulación.
